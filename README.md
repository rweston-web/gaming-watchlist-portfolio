## 🕹️ What This Project Is About

I am a web developer who loves gaming, so I built a dark, cyberpunk-themed website with neon-styled text that holds my real-world projects across three separate pages:

1. **Home Page:** A clean introduction about who I am, a breakdown of how I built this site, and a working Contact Form that sends messages directly to my email inbox.
2. **Gaming Watchlist (`watchlist-page.js`):** A custom tracker for my favorite games like. I typed the JavaScript click tracking code to handle each game card independently. When you click a button, it instantly changes the text and switches the style to bright green without reloading the whole page.
3. **RPG D20 Dice Roller (`dice-roller.js`):** A digital 20-sided die calculator built for tabletop games. When you hit the roll button, the code runs a high-speed random number shuffle animation. Once it lands, the script checks the number and flashes custom battle alerts on screen—including a bright green text explosion for a "Natural 20 Critical Hit!" or a red warning for a "Critical Fail!"
